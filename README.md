# 📊 E-Commerce Sales Analysis & Interactive Dashboard

A professional-grade data analytics project demonstrating the full ETL (Extract, Transform, Load) process—from querying raw SQL data to designing a custom UI in Figma and building an interactive dashboard in Power BI.

## 🚀 Project Overview
This project analyzes e-commerce sales performance to identify key revenue drivers, top-performing categories, and high-value customers. 

**Key Insights:**
* **Top Category:** Clothing (dominates total revenue).
* **Top Customer:** Customer_24 (highest individual contribution).
* **Trends:** Monthly fluctuations indicate seasonal purchasing patterns.

## 🛠️ Tech Stack
* **Database:** SQL (Data extraction & cleaning)
* **Design:** Figma (Custom UI/UX background)
* **Visualization:** Power BI
* **Data Source:** Kaggle

## 🎨 Design Philosophy
Unlike standard "out-of-the-box" templates, this dashboard uses a **custom-designed Figma UI**. The "Dark Mode" aesthetic with red gradients was chosen to improve readability of KPIs and provide a premium, executive-level feel.

## 📈 Dashboard Features
* **KPI Cards:** Real-time tracking of Total Revenue, Orders, and Customers.
* **Monthly Trend:** Line chart showing growth and drops over the fiscal year.
* **Category Breakdown:** Bar chart identifying the most profitable product lines.
* **Interactive Slicers:** Filter the entire report by  **Category** for deep-dive analysis.

## 📂 How to Use
1. Clone the repo.
2. Run the scripts in `/SQL_Queries` to see the data logic.
3. Open the `.pbix` file in `/Dashboard` to interact with the visuals.
